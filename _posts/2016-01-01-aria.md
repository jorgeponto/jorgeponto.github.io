Em HTML podemos definir os diversos elementos em 2 grupos: (a) elementos semânticos ou com um papel originalmente definido; (b) elementos genéricos, sem papel original atribuído.

## Elementos semânticos

Por exemplo, o elemento ´<p>´ representa, ou tem como papel atribuído, um parágrafo. O elemento ´<abbr>´ representa uma abreviatura ou acrónimo - Em HTML5 o acrónico ou a abreviatura são marcados pelo elemento ´<abbr>´. O elemento parágrafo e o elemento abreviatura têm originalmnete na linguagem HTML um papel definido - o texto que é colocado no seu interior ganha de imediato um significado.

A cada nova versão das especificações técnicas da linguagem de marcação de HTML surgem novos elementos e há elementos que desaparecem tornando-se obsoletos. Por exemplo, no HTML5 o elemento '<acronym>´ foi um dos que desapareceu tornado-se obsoleto. Geralmente, quando um elemento se torna obsoleto é porque existem outras hipóteses para o substituir na nova especificação. No caso do HTML5 passou-se a usar o elemento ´<abbr>´ para marcar quer as abreviaturas quer os acrónimos.

## Elementos genéricos

Em HTML há 2 elementos genéricos, ou seja que não atribuem originalmente um significado ao texto que estão a marcar. São eles: o elemento ´<div>´ e o elemento ´<span>´.
