Este é o conteúdo original

Se passarmos este conteúdo pelo validador de acccessibilidade AccessMonitor, ele vai ver um link "Accessmonitor" na página. Depois de carregarmos no botão "Alterar conteúdo" e passarmos a página pelo mesmo validador, ele não vai ver o link. Se tivêssemos usado técnicas de CSS para o fazer, visualmente teríamos o mesmo efeito mas o validador iria ver o link mesmo quando este não é visível no ecrã.
